# Camadas e Responsabilidades

<b>- 1 - APIs:</b> Web API que atende requisições feitas pelo Web App;
<b>- 2 - Workers:</b> Worker Services que processa dados em backgroud via AMQP com RabbitMQ.
<b>- 3 - Shared:</b> Camada compartilhada entre projetos.
<b>- 4 - Business:</b> Camada com as regras de negócios da aplicação.
<b>- 5 - Data:</b> Camada de acesso a dados.
<b>- 6 - Tests:</b> Camada de testes de unidades.
